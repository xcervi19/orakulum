Název kroku: Pokročilé kompoziční systémy  
Hlavní záměr: Vytvářet vizuály s vysokou estetickou i funkční přesností.

Obsah kroku:

* Kompozice je architektura pozornosti. Zatímco nováčci „skládají prvky“, profesionálové používají kompoziční sítě jako matematické rozhraní mezi estetikou a výkonem. Rule of thirds, golden ratio, dynamic symmetry a typografická mřížka nejsou pravidla, ale instrukční systémy, které určují, kde vzniká napětí, rovnováha nebo dominance.
* Příklady:
  • Rule of thirds: ideální pro rychle čitelné reklamní vizuály – CTA lze umístit na průsečík, aby přitahoval oko.
  • Golden ratio: funguje u prémiových značek; spirála vede oko přes příběh produktu.
  • Dynamic symmetry (root rectangles): vhodné pro motion vizuály nebo TikTok hooky, kde je třeba dynamika místo statické rovnováhy.
  • Typografická mřížka: klíčová pro emaily, landing pages a karusely – čitelnost roste s pevnou hierarchií.
* Případová studie:
  Firma měla problém s nízkým scroll-stop rate. Původní vizuál byl kompozičně těžký vlevo. Redesign pomocí dynamic symmetry přesunul produkt do diagonálního tahu a CTA na harmonický bod root-3 obdélníku. Výsledek: +22 % scroll-stop rate.
* Typické pohovorové otázky a optimální odpovědi:
  • „Jak volíte kompoziční systém?“ → „Podle povahy sdělení: brand = golden ratio, performance = thirds, dynamický obsah = dynamic symmetry, textové layouty = grid.“
  • „Jak kompozice ovlivňuje výkon kampaně?” → „Ovlivňuje čtecí rytmus, prioritu informací a rychlost pochopení. Správná kompozice zkrátí kognitivní load a zvedá engagement.“
  • „Jak pracujete s typografickou mřížkou?“ → „Definuji baseline grid, modulární systém a kontroluji aligny pro konzistenci a rychlou čitelnost.“
* Mini úkoly:
  • Vezmi dva stávající vizuály a přepracuj je: jeden pomocí golden ratio, druhý pomocí typografické mřížky.
  • Pro každý redesign napiš 3 věty vysvětlující, proč zvolený systém zlepšuje performance metriky.
* Návaznost na krok 10: Analytická přesnost ti ukázala, *co funguje*. Kompoziční systémy ti dávají kontrolu nad tím, *proč* to funguje — a jak to dělat opakovaně.

Praktická část:

```
# Algoritmus pro výpočet zlatého řezu (phi) a návrh spirály pro kompoziční vodítka
phi = (1 + 5**0.5) / 2

def golden_rect(w):
    h = w / phi
    return (w, h)

def subdivide(w, h, steps=4):
    rects = []
    for _ in range(steps):
        if w > h:
            w = w - h
            rects.append((w, h))
        else:
            h = h - w
            rects.append((w, h))
    return rects

print(subdivide(1000, 618))
```

Diagram kompozičních systémů:
THIRDS = stabilita
GOLDEN = elegance
DYNAMIC SYMMETRY = energie
GRID = racionalita

Ukazatele úspěchu:

* Umíš vědomě volit kompoziční princip podle typu sdělení.
* Redesign dvou vizuálů pomocí dvou různých systémů je jasně obhajitelný.
* Dokážeš vysvětlit kompoziční volby u pohovoru jako profesionální rozhodnutí, ne jako intuici.
* Tvé vizuály mají konzistentně vyšší čitelnost, rovnováhu a predikčně kvalitnější metriky.

Bonus / laterální rozšíření:
- Aplikuj hudební rytmus na vizuální layout: crescendo = postupné navyšování velikosti prvků; kontrapunkt = paralelní, ale vyvážené linie textu vs. obrazu; pauza = negativní prostor. Hudební kompozice a vizuální kompozice sdílejí stejnou strukturu „napětí → uvolnění“. Přenes tuto logiku do layoutů a sleduj, jak se zlepší jejich narativní proud i emocionální rezonance.
