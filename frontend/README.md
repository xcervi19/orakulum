# Frontend Onboarding

This folder contains the refactored onboarding UI.

## Setup

1. Open `index.html`
2. Update `SUPABASE_KEY` with your project's **Anon Key**.
3. Update `SUPABASE_URL` if different from the one in `database_schema.sql`.

## Features

- Stepper flow (Goals, Area, Level, etc.)
- Live Activity Panel (simulated activity)
- Progress bar
- Supabase integration (inserts into `junior_leads`)

## Deployment

Deploy this `index.html` to any static hosting (Netlify, Vercel, GitHub Pages).
