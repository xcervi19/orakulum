  <h2 data-ui="heading">Název kroku: Mikro-specializace (1. úroveň)</h2>
  <p data-ui="paragraph">Hlavní záměr: Vybrat úzkou subdoménu, ve které lze rychle růst, získat jasnou identitu a produkovat konzistentní hodnotu pro trh i zaměstnavatele.</p>

  <h3 data-ui="heading">Obsah kroku:</h3>

  <ul data-ui="bullet-list">
    <li data-ui="list-item">
      <p data-ui="label">detailní vysvětlení tématu</p>
      <p data-ui="paragraph">Mikro-specializace určuje, jaký typ problémů řešíte nejlépe: performance design, branding, social content nebo motion. Cílem je vytvořit profesionální fokus, který zrychlí učení, zvýší kvalitu výstupů a především vás jasně odliší u pohovoru. Recruiter vidí: *„Tento člověk je expert na X a má 15 konzistentních prací.“*</p>
    </li>

```
<li data-ui="list-item">
  <p data-ui="label">příklady</p>
  <ul data-ui="bullet-list">
    <li data-ui="list-item">
      <p data-ui="paragraph">Performance designer se zaměřuje na strukturu vizuálu, čitelnost CTA, rychlost pochopení sdělení.</p>
    </li>
    <li data-ui="list-item">
      <p data-ui="paragraph">Branding designer řeší systém, konzistenci, gridy, logiku vizuální identity.</p>
    </li>
    <li data-ui="list-item">
      <p data-ui="paragraph">Social content specialista pracuje s variabilitou, mikro-storytellingem a rytmem.</p>
    </li>
    <li data-ui="list-item">
      <p data-ui="paragraph">Motion specialista tvoří animace, přechody, krátké dynamické spoty.</p>
    </li>
  </ul>
</li>

<li data-ui="list-item">
  <p data-ui="label">případové studie nebo modelové situace</p>
  <ul data-ui="bullet-list">
    <li data-ui="list-item">
      <p data-ui="paragraph">Freelancer vytvořil mikro-portfólio 12 výkonnostních bannerů pro 3 segmenty (fitness, SaaS, fashion). Za 6 týdnů získal stabilní kontrakt, protože klient jasně viděl, že *umí řešit specifický problém*, ne „všechno trochu“.</p>
    </li>
    <li data-ui="list-item">
      <p data-ui="paragraph">Junior motion designer vytvořil 10 micro-animací pro hypotetickou značku. V pohovoru se stal top uchazečem díky tomu, že jeho styl byl konzistentní a použitelný v reálné produkci.</p>
    </li>
  </ul>
</li>

<li data-ui="list-item">
  <p data-ui="label">typické pohovorové otázky a optimální odpovědi</p>
  <ul data-ui="bullet-list">
    <li data-ui="list-item">
      <div data-ui="qa-block">
        <p data-ui="question">*Jakou oblast designu chcete rozvíjet a proč?*</p>
        <p data-ui="answer">„Soustředím se na performance design, protože mě baví kombinovat psychologii pozornosti, jasnou výzvu k akci a rychlé testování. Tvořím vizuály, které mají měřitelný dopad.“</p>
      </div>
    </li>
    <li data-ui="list-item">
      <div data-ui="qa-block">
        <p data-ui="question">*Jak přistupujete k tvorbě obsahu pro různé cílové skupiny?*</p>
        <p data-ui="answer">„Začnu mikro-personou → definice motivace → volba stylu → test 3 verzí. Mám systém, ne náhodný výběr.“</p>
      </div>
    </li>
    <li data-ui="list-item">
      <div data-ui="qa-block">
        <p data-ui="question">*Jak vnášíte konzistenci do vizuálů?*</p>
        <p data-ui="answer">„Používám pevně definovaný stylový rámec: kontrastní páry, typografickou hierarchii, opakující se motivy a grid.“</p>
      </div>
    </li>
  </ul>
</li>

<li data-ui="list-item">
  <p data-ui="label">mini úkoly nebo praktická cvičení</p>
  <ul data-ui="tasks">
    <li data-ui="task">
      <p data-ui="paragraph">Vytvořte 5 reklamních vizuálů pro 5 různých cílovek (mladé mámy, early adopters, studenti, profesionálové 30+, fanoušci luxusu).</p>
    </li>
    <li data-ui="task">
      <p data-ui="paragraph">Vyberte jeden styl, kterému dáte jasná pravidla: barvy, rytmus, typografie, kompozice, tón CTA.</p>
    </li>
    <li data-ui="task">
      <p data-ui="paragraph">Ověřte, zda jsou vizuály „scanovatelné“ do 2 sekund — quick test: zakryjte 70 % vizuálu a sledujte, co zůstane zapamatovatelné.</p>
    </li>
  </ul>
</li>

<li data-ui="list-item">
  <p data-ui="label">návaznost na krok n-1</p>
  <p data-ui="paragraph">Navazuje na fundamentální kompetence z kroku 2 — nyní se nejedná jen o pochopení principů, ale o jejich cílené uplatnění v jedné subdoméně. „Základní dovednosti → ostřený fokus → portfolio.“</p>
</li>
```

  </ul>

  <h3 data-ui="heading">Praktická část:</h3>

  <ul data-ui="bullet-list">
    <li data-ui="list-item">
      <p data-ui="label">Strukturovaný problém: Navrhněte 3 vizuály pro stejný produkt, ale odlišné subdomény.</p>
      <ol data-ui="numbered-list">
        <li data-ui="list-item">
          <p data-ui="paragraph">Performance varianta (ASCII wireframe):</p>
          <pre data-ui="code-block" data-language="algorithm"><code>```
[PRODUKT FOTO]  
[BOLD BENEFIT]  
[CTA]  
```</code></pre>
        </li>
        <li data-ui="list-item">
          <p data-ui="paragraph">Branding varianta (grid):</p>
          <pre data-ui="code-block" data-language="algorithm"><code>```
|LOGO|---TAGLINE---|  
|COLOR BLOCK|IMAGE|  
|CTA|  
```</code></pre>
        </li>
        <li data-ui="list-item">
          <p data-ui="paragraph">Social content varianta (krátký koncept):</p>
          <p data-ui="paragraph">“3-sekundový mikro-příběh → punchline → CTA”</p>
        </li>
      </ol>
    </li>
    <li data-ui="list-item">
      <p data-ui="label">Mini-algoritmus výběru specializace:</p>
      <pre data-ui="code-block" data-language="algorithm"><code>```
score = (radost_z_tvorby * 0.5) + (rychlost učení * 0.3) + (poptávka_na_trhu * 0.2)
vyber_subdomenu(score.max())
```</code></pre>
    </li>
    <li data-ui="list-item">
      <p data-ui="label">Mini-test: Dokážeš vytvořit 5 vizuálů, z nichž je na první pohled jasné, že patří do jedné stylistické rodiny?</p>
    </li>
  </ul>

  <h3 data-ui="heading">Ukazatele úspěchu:</h3>

  <ul data-ui="metric-list">
    <li data-ui="metric">
      <p data-ui="paragraph">vzniká micro-portfólio 10–15 prací v jednom konkrétním stylu</p>
    </li>
    <li data-ui="metric">
      <p data-ui="paragraph">konzistentní vizuální jazyk napříč projekty</p>
    </li>
    <li data-ui="metric">
      <p data-ui="paragraph">připravené ukázky, které lze u pohovoru prezentovat jako „micro-expertise package“</p>
    </li>
    <li data-ui="metric">
      <p data-ui="paragraph">samostatné argumentování: *proč jsem si vybral tuto subdoménu a jak v ní rostu*</p>
    </li>
  </ul>

  <h3 data-ui="heading">Bonus / laterální rozšíření:</h3>
  <ul data-ui="bullet-list">
    <li data-ui="list-item">
      <p data-ui="paragraph">- Experiment „design in constraints“: vytvoř 3 kampaně pod extrémními omezeními.</p>
      <ul data-ui="bullet-list">
        <li data-ui="list-item">
          <p data-ui="paragraph">- pouze černobílá paleta (disciplína kontrastu)</p>
        </li>
        <li data-ui="list-item">
          <p data-ui="paragraph">- bez textu (čistá vizuální srozumitelnost)</p>
        </li>
        <li data-ui="list-item">
          <p data-ui="paragraph">- pouze text (typografická síla, rytmus, tón)</p>
        </li>
      </ul>
      <p data-ui="paragraph">  Tento typ cvičení rozvíjí laterální kreativitu, odstraňuje závislost na „bezpečných“ řešeních a výrazně vylepšuje schopnost improvizace — což je jedna z nejvíce oceňovaných kompetencí u kreativních pohovorů.</p>
    </li>
  </ul>
