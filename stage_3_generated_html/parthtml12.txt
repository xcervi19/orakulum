  <h2 data-ui="heading">Název kroku: Mastery typografie</h2>
  <p data-ui="paragraph">Hlavní záměr: Převést typografii z dekorace na nástroj řízení pozornosti.</p>

  <h3 data-ui="heading">Obsah kroku:</h3>

  <ul data-ui="bullet-list">
    <li data-ui="list-item">
      <p data-ui="paragraph">Typografie je nejpřesnější forma vizuálního jazyka. Není to ozdoba, ale matematika čitelnosti, psychologie mikro-gest a nástroj, kterým určuješ kognitivní tok. Kerning a tracking ovlivňují tempo čtení, kontrast vah textu vede oko, baseline grid dává rytmus a variable fonts umožňují tvořit vlastní nuance, které odlišují značku.</p>
    </li>
    <li data-ui="list-item">
      <p data-ui="label">Příklady:</p>
      <ul data-ui="bullet-list">
        <li data-ui="list-item">
          <p data-ui="paragraph">• Kerning: Při špatném kerningu se zkracují „optické mezery“ a čtení zpomaluje. U CTA může rozdíl 10 jednotek kerningu ovlivnit pochopení do 200 ms.</p>
        </li>
        <li data-ui="list-item">
          <p data-ui="paragraph">• Tracking: Zvýšení tracking o +5 % u long-form textů zlepšuje čitelnost, ale u claimů snižuje vizuální sílu.</p>
        </li>
        <li data-ui="list-item">
          <p data-ui="paragraph">• Kontrast vah: Bold → přitáhne; Regular → stabilizuje; Light → vytváří vzdušnost. Hierarchie vah je rychlejší než barva.</p>
        </li>
        <li data-ui="list-item">
          <p data-ui="paragraph">• Variable fonts: Umožňují doladit variantu mezi 450–520 weight, což je zóna ideální pro výkonné digital ads.</p>
        </li>
      </ul>
    </li>
    <li data-ui="list-item">
      <p data-ui="label">Případová studie:</p>
      <p data-ui="paragraph">Výkonnost banneru stagnovala. Analýza ukázala, že headline má weight 300 a příliš jemný kontrast vůči pozadí. Změna na weight 520 (variable font), úprava kerningu -1 %, CTA zvýrazněno na semi-bold. Výsledek: CTR +14 %, time-to-read -0,4 s.</p>
    </li>
    <li data-ui="list-item">
      <p data-ui="label">Typické pohovorové otázky a optimální odpovědi:</p>
      <ul data-ui="bullet-list">
        <li data-ui="list-item">
          <p data-ui="question">• „Jak pracujete s baseline gridem?“</p>
          <p data-ui="answer">→ „Definuji vertikální rytmus a zarovnání všech textových prvků, aby vznikla předvídatelná struktura. Zlepšuje to čitelnost i vizuální autoritu značky.“</p>
        </li>
        <li data-ui="list-item">
          <p data-ui="question">• „Jak poznáte, že typografie je výkonově správně nastavena?“</p>
          <p data-ui="answer">→ „Claim čitelný do 1 s, CTA odlišitelné v periferním vidění, konzistentní vizuální váhy napříč formáty.“</p>
        </li>
        <li data-ui="list-item">
          <p data-ui="question">• „Proč jsou variable fonts důležité?“</p>
          <p data-ui="answer">→ „Umožňují mikro-optimalizace pro různé kanály a zvyšují adaptabilitu brandu bez vizuální nekonzistence.“</p>
        </li>
      </ul>
    </li>
    <li data-ui="list-item">
      <p data-ui="label">Mini úkoly:</p>
      <ul data-ui="bullet-list">
        <li data-ui="list-item">
          <p data-ui="task">• Vytvoř 3 typografické systémy: (1) fintech – precizní, úřední, vysoký kontrast; (2) fashion – elegantní, vzdušný; (3) tech startup – moderní, vysoký rytmus, variabilní váhy.</p>
        </li>
        <li data-ui="list-item">
          <p data-ui="task">• Každému systému nastav: kerning pattern, tracking policy, váhovou hierarchii, baseline strategii.</p>
        </li>
      </ul>
    </li>
    <li data-ui="list-item">
      <p data-ui="label">Návaznost na krok 11:</p>
      <p data-ui="paragraph">V kroku 11 ses naučil, jak pracovat s kompozičními sítěmi. Zde ovládáš jejich nejjemnější vrstvu — textové mikrosystémy, které určují přesnost vizuálního toku.</p>
    </li>
  </ul>

  <h3 data-ui="heading">Praktická část:</h3>

  <pre data-ui="code-block" data-language="algorithm"><code># Generátor typografické hierarchie pomocí variable fontů
base_weight = 450
hierarchy = {
    "headline": base_weight + 70,
    "subhead": base_weight + 20,
    "body": base_weight,
    "cta": base_weight + 90
}

for level, weight in hierarchy.items():
    print(level, weight)</code></pre>

  <h3 data-ui="heading">Strukturovaný problém:</h3>

  <ol data-ui="numbered-list">
    <li data-ui="list-item">
      <p data-ui="task">Vyber claim.</p>
    </li>
    <li data-ui="list-item">
      <p data-ui="task">Vytvoř 5 typografických variant – měň jen váhu, kerning a tracking.</p>
    </li>
    <li data-ui="list-item">
      <p data-ui="task">Otestuj 1sekundové pochopení u 3 lidí.</p>
    </li>
    <li data-ui="list-item">
      <p data-ui="task">Spočítej, která varianta měla nejvyšší <em>clarity index</em> (subjektivně i měřitelně).</p>
    </li>
  </ol>

  <h3 data-ui="heading">Ukazatele úspěchu:</h3>

  <ul data-ui="bullet-list">
    <li data-ui="list-item">
      <p data-ui="metric">Typografie sama o sobě vytvoří jasné sdělení bez obrázků.</p>
    </li>
    <li data-ui="list-item">
      <p data-ui="metric">Dokážeš obhájit volbu vah, kerningu i gridu jako výkonovou, ne estetickou.</p>
    </li>
    <li data-ui="list-item">
      <p data-ui="metric">Tvé typografické systémy jsou konzistentní napříč kanály i formáty.</p>
    </li>
    <li data-ui="list-item">
      <p data-ui="metric">Při pohovoru umíš vysvětlit, jak typografie zrychluje porozumění a zvyšuje konverze.</p>
    </li>
  </ul>
