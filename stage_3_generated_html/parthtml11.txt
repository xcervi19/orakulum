  <h2 data-ui="heading">Název kroku: Pokročilé kompoziční systémy</h2>
  <p data-ui="paragraph">Hlavní záměr: Vytvářet vizuály s vysokou estetickou i funkční přesností.</p>

  <h3 data-ui="heading">Obsah kroku:</h3>

  <ul data-ui="bullet-list">
    <li data-ui="list-item">
      <p data-ui="paragraph">Kompozice je architektura pozornosti. Zatímco nováčci „skládají prvky“, profesionálové používají kompoziční sítě jako matematické rozhraní mezi estetikou a výkonem. Rule of thirds, golden ratio, dynamic symmetry a typografická mřížka nejsou pravidla, ale instrukční systémy, které určují, kde vzniká napětí, rovnováha nebo dominance.</p>
    </li>

```
<li data-ui="list-item">
  <p data-ui="label">Příklady:</p>
  <ul data-ui="bullet-list">
    <li data-ui="list-item">
      <p data-ui="paragraph">• Rule of thirds: ideální pro rychle čitelné reklamní vizuály – CTA lze umístit na průsečík, aby přitahoval oko.</p>
    </li>
    <li data-ui="list-item">
      <p data-ui="paragraph">• Golden ratio: funguje u prémiových značek; spirála vede oko přes příběh produktu.</p>
    </li>
    <li data-ui="list-item">
      <p data-ui="paragraph">• Dynamic symmetry (root rectangles): vhodné pro motion vizuály nebo TikTok hooky, kde je třeba dynamika místo statické rovnováhy.</p>
    </li>
    <li data-ui="list-item">
      <p data-ui="paragraph">• Typografická mřížka: klíčová pro emaily, landing pages a karusely – čitelnost roste s pevnou hierarchií.</p>
    </li>
  </ul>
</li>

<li data-ui="list-item">
  <p data-ui="label">Případová studie:</p>
  <p data-ui="paragraph">Firma měla problém s nízkým scroll-stop rate. Původní vizuál byl kompozičně těžký vlevo. Redesign pomocí dynamic symmetry přesunul produkt do diagonálního tahu a CTA na harmonický bod root-3 obdélníku. Výsledek: +22 % scroll-stop rate.</p>
</li>

<li data-ui="list-item">
  <p data-ui="label">Typické pohovorové otázky a optimální odpovědi:</p>
  <ul data-ui="bullet-list">
    <li data-ui="list-item">
      <p data-ui="question">• „Jak volíte kompoziční systém?“</p>
      <p data-ui="answer">→ „Podle povahy sdělení: brand = golden ratio, performance = thirds, dynamický obsah = dynamic symmetry, textové layouty = grid.“</p>
    </li>
    <li data-ui="list-item">
      <p data-ui="question">• „Jak kompozice ovlivňuje výkon kampaně?”</p>
      <p data-ui="answer">→ „Ovlivňuje čtecí rytmus, prioritu informací a rychlost pochopení. Správná kompozice zkrátí kognitivní load a zvedá engagement.“</p>
    </li>
    <li data-ui="list-item">
      <p data-ui="question">• „Jak pracujete s typografickou mřížkou?“</p>
      <p data-ui="answer">→ „Definuji baseline grid, modulární systém a kontroluji aligny pro konzistenci a rychlou čitelnost.“</p>
    </li>
  </ul>
</li>
```

  <h3 data-ui="heading">Praktická část:</h3>

  <pre data-ui="code-block" data-language="algorithm"><code># Algoritmus pro výpočet zlatého řezu (phi) a návrh spirály pro kompoziční vodítka
phi = (1 + 5**0.5) / 2

def golden_rect(w):
    h = w / phi
    return (w, h)

def subdivide(w, h, steps=4):
    rects = []
    for _ in range(steps):
        if w &gt; h:
            w = w - h
            rects.append((w, h))
        else:
            h = h - w
            rects.append((w, h))
    return rects

print(subdivide(1000, 618))</code></pre>
